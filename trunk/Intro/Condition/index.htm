1.教材使用与建设


     自2004年开设《VB程序设计》课程以来，使用由罗朝盛主编、郑玲利副主编的《Visual Basic 6.0程序设计使用教程》（清华大学出版社2004年）。从2007年开始使用此教材的第二版。该教材属于高等学校计算机基础教育教材精选，具有相当高的教学和学术水平。目前，针对各版教材中不适合的部分，教学队伍正在自行组织材料编写教材，以达到最佳的教学效果。
 

2.学生主动学习的扩充性资料使用情况


    自2004年起，教学队伍相继制作了电子课件、电子教案、电子课程大纲、电子实验大纲、以及网上题库。采用现代化教学手段，全面提高教学效果和学生动手能力，充分实现教学资源共享。同时，学校图书馆提供有大量的纸质和电子图书、期刊杂志，中外文数据库向学生开放，为学生自主性学习提供了丰富的资料。

3.配套实验教材的教学效果

  实验教材为《Visual Basic 6.0程序设计使用教程》，教学队伍按照任务驱动的教学方法，认真做好指导学生实验，使学生通过上机操作掌握本课程所讲述的程序设计思想，并能独立编写VB应用程序，为提高学生的实际编程能力打下基础。

4. 实践性教学环境

  为保证计算机基础教育有一个良好的上机实验环境，帮助学生运用Visual Basic程序设计语言解决本专业简单应用问题。自2004年以来，学校投入大量资金，用于购置微型计算机、服务器等实验设备和相关的系统软件与应用软件，改造了机房的网络系统，完善了计算机基础实验室的实验条件，为学生提供了完备的上机实验环境。目前，VB实验室使用面积超过100平米，拥有高档微型计算机40余台、四核Xeon服务器一台、1000M光纤/电口模块交换机一台、两千瓦不间断电源UPS一台、60点的LANSTAR教学软件一套等先进的软硬件设备。实验室单独组成局域网，且每一台计算机都连接校园网。如下图所示：

5.网络教学环境

目前《VB程序设计》课程网站还在制作中。将来，该课程网站将包含的主要栏目有：课程简介、师资队伍、教学资料、实验指导、网络课程、同步练习、在线测试、习作园地、常见问题、下载中心、我要留言等，内容丰富，形式灵活多样。
