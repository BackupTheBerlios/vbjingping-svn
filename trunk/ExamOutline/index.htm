一、课程的性质与设置目的

(一)课程性质和特点

《Visual Basic程序设计》课程是一门专业选修课。本课程主要讲授Microsoft 公司的可视化编程工具Visual Basic。通过本课程的学习，掌握使用Visual Basic进行Windows应用程序设计的方法，可以熟练运用窗体和控件进行界面设计，编制并调试程序，同时要掌握程序设计的思想与方法,培养良好的程序设计素养与规范的程序编码方法。 为各专业后继相关课程打下计算机程序设计理论与实践扎实的基础。

(二)本课程的基本要求

本课程教学的基本要求：
  • 使学生掌握VB相关的程序设计语言基础；
  • 掌握面向对象编程的基本知识和开发应用程序的基本能力
  • 掌握使用VB开发应用程序的方法，熟练使用窗体和标准控件；
  • 通过VB数据库操作与编程掌握管理系统实现的相关技术；
  • 通过图形编程等掌握一定的多媒体开发技巧；

    • 通过本课程的学习，使学生掌握用VB编程解决实际问题的一般方法，包括语句、函数、控件等概念的掌握，以及从基本控件的使用到高级控件的应用，从图形编程、文件使用到多媒体程序开发、数据库编程等内容，为其它编程工具的学习以及成为一名熟练的程序员打下坚实的基础。

    (三)本课程与相关课程的联系

    本课程的先修课程：计算机基础、程序设计基础

    二、课程内容与考核目标

    第1章 绪论

    (一)课程内容
    1.1 VB语言特点
    1.2 VB 6.0的功能、特点
    1.3 VB 6.0的新功能特点
    1.4 安装VB6.0
    1.5 Visual Basic 6.0可视化集成开发环境
    1.6简单的VB用户程序建立的步骤、运行和存储
    1.7 对象的概念、对象的属性、事件和方法
    (二)考核目的与要求
    要求掌握VB开发环境和面向对象程序开发要掌握的概念。

    第2章 Visual Basic的窗体及基本控件

    (一)课程内容
    2.1 基本属性
    2.2 窗体和其属性、事件、方法
    2.3 标签和其属性、事件、方法
    2.4文本框和其属性、事件、方法
    2.5 命令按钮和其属性、事件、方法
    (二) 考核目的与要求
    本章要求掌握Visual Basic用户界面设计中的窗体、标签、文本框和命令按钮的相关属性设置、方法和事件，以及控件的基本属性设置。

    第3章 Visual Basic 6.0语言基础

    (一)课程内容
    3.1数据类型、变量和常量
    3.2 表达式和运算符、函数
    3.3 基本语句
    3.4 程序错误
    3.5 程序调试
    3.6 错误捕获及处理
    3.7 条件编译
    (二) 考核目的与要求
    本章要求掌握VB程序编写中的相关知识，包括变量、常量的定义，函数和表达式的书写，基本语句格式要求等内容，同时能够调试程序。

    第4章 数组

    （一）课程内容
    4.1 静态数组
    4.2 动态数组
    4.3 控件数组
    （二）考核目的与要求
    本章要求掌握数组的定义、赋值、输出等相关操作，包括动态和静态数组，以及控件数组的建立和编程。

    第5章 过程和函数

    （一）课程内容
    5.1 过程
    5.2 函数
    5.3 参数的传递
    5.4 变量过程的作用域
    （二）考核目的与要求
    本章要求掌握VB中过程函数的声明、调用，参数的按值、按址两种传递方式及全局变量、窗体或模块级变量、局部变量、静态变量的概念和作用域。
